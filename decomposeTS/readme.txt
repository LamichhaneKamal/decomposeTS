This is a higly generic temporal sequence decomposition algorithm, which is 
applicable to continuous/discrete uni-variate/multi-variate time series.

Please cite the paper if using the code:
@inproceedings{zhao2016decomposing,
  title={Decomposing Time Series With Application to Temporal Segmentation},
  author={Zhao, Jiaping and Itti, Laurent},
  booktitle={Proceedings of the IEEE Winter Conference on Applications of Computer Vision (WACV)},
  year={2016}
}



How to run the code:
(1) run "config.m" first to setup paths;
(2) run demo examples "demo_decomposeTS.m"
